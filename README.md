# 📝 How to Complete an Assignment

## 1️⃣ Accept the Assignment

- Go to the GitHub Classroom link: https://classroom.github.com/a/dgZEgqCw 
- Click **Accept Assignment**.
- GitHub will create your own private repository copy.

---

## 2️⃣ Clone Your Repository

Copy your repository URL and run:

```bash
git clone <your-repository-url>
Then move into the folder:

cd your-repository-name
3️⃣ Open in VS Code
code .
Navigate to the correct week/day/assignment folder:

Example:

week-07/day-01/assignment-01/
4️⃣ Complete the Assignment
Write your HTML/CSS/JS inside the correct folder.

Do NOT rename folders.

Do NOT move files outside your assignment folder.

💾 How to Submit (Push Your Work)
After completing your assignment:

1️⃣ Stage Changes
git add .
2️⃣ Commit
git commit -m "Complete Week 07 Day 01 Assignment 01"
3️⃣ Push
git push
Your work is now submitted automatically.

📌 Important Rules
Push your work before the deadline.

Commit frequently.

Keep clean folder structure.

Do not delete required folders.

Ask questions early if you get stuck.
